# Antifraud Service API

 Сервис анализирует данные пользователя и возвращает результат проверки с указанием стоп-факторов.

##  Технологии

- **FastAPI** - современный веб-фреймворк
- **Redis** - высокопроизводительное кэширование
- **Pydantic** - строгая валидация данных
- **Docker** - контейнеризация приложения
- **pytest** - комплексное тестирование

##  Функционал

- **Валидация возраста** - проверка,что пользователь старше 18 лет  
- **Проверка номера телефона** - только российские номера  
- **Анализ займов** - обнаружение открытых займов  
- **Кэширование** - результаты хранятся в Redis  
- **Автотесты** - проверка правильности функций 
- **Docker готовность** - полная контейнеризация

 API Endpoints

# POST /users/ - Проверка пользователя

Проверяет пользователя на стоп-факторы и возвращает результат.

Request:
json

{
  "birth_date": "15.05.1985",
  "phone_number": "+79161234567",
  "loans_history": [
    {
      "amount": 10000,
      "loan_data": "01.01.2023",
      "is_closed": true
    }
  ]
}

Response:
json

{
  "stop_factors": [],
  "result": true
}
